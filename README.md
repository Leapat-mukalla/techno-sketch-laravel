# Developer Guide
### RUN PROJECT:
```bash
php artisan serve
```
### CONFIGURE DB:
```bash
php artisan migrate
```
```bash
php artisan db:seed
```
## Dependencies
- PHP
- Composer
- Mysql
- Redis
